60000 and 70000 .

<h3>Query 4B</h3>

<h2>***Query 5***</h2>

department.
<h3>Query 5B</h3>

<h3>Query 5C</h3>
Label the computed columns in the result set of Query 5B as “NUM_EMPLOYEES” and “AVG_SALARY”.
<h3>Query 5D</h3>
In Query 5C order the result set by Average Salary.
<h3>Query 5E</h3>
